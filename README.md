# Email Spam Classification Project

This repository contains an end-to-end machine learning project for classifying emails or SMS as spam or ham using the UCI Spam Dataset. The application includes data preprocessing, model training with algorithms like Logistic Regression, Naive Bayes, and SVM, and deployment as a Flask web app on Heroku.
